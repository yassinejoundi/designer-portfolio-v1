## Live Link

You can check the live version [Here](https://designer-portfolio-v1.vercel.app)

## Getting Started

```bash
# clone the project
$ git clone https://github.com/yassinejoundi/designer-portfolio-v1.git
# install dependencies
$ npm install
# serve with hot reload at localhost:3000
$ npm run dev
# build for production and launch server
$ npm run build
$ npm run start
# generate static project
$ npm run generate
```

if you don't have Vuejs installed

```bash
$ npm install -g @vue/cli
```
